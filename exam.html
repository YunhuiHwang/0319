모르면 마법, 알면 기술이다.

김민준의 기술 블로그에 오신 여러분을 환영합니다.

대학 졸업을 앞둔 컴퓨터 공학 전공의 학생, 김민준입니다. 전공 수업과 여러 프로젝트를 통해 소프트웨어 개발에 필요한 다양한 프로그래밍 언어와 기술을 익혔습니다. 특히, 인공 지능과 모바일 애플리케이션 개발에 깊은 관심을 가지고 있습니다. 대학교에서는 동아리 활동을 통해 팀워크와 리더십 경험도 쌓았습니다. 졸업 후에는 이러한 기술과 경험을 바탕으로 IT 분야에서 전문 개발자로 성장하고자 합니다.

웹 개발에 관심이 많아, 프론트엔드 개발에 필요한 여러 기술을 스스로 학습하고 있습니다. 제가 주로 사용하는 프론트엔드 기술은 HTML, CSS, JavaScript입니다. 이를 기반으로 최근에는 React.js 프레임워크를 배우기 시작했습니다. React를 통해 컴포넌트 기반의 개발 방식과 상태 관리의 중요성을 이해하게 되었고, 이를 실제 프로젝트에 적용해 보았습니다. 또한, Redux를 사용하여 복잡한 상태 관리를 경험했으며, 이 과 정에서 불변성을 유지하는 방법과 효율적인 데이터 흐름을 설계하는 방법을 배웠습니다. 프로젝트를 진행하면서 CSS 프레임워크인 Bootstrap과 Material-UI를 사용해 보았습니다. 이 도 구들은 반응형 디자인을 손쉽게 구현할 수 있게 해주어 사용자 경험을 크게 향상시키는 데 기여 했습니다. 이러한 경험을 통해 사용자 중심의 UI/UX 디자인의 중요성을 더 깊이 이해하게 되었 습니다.

•수상경력: 구글에서 주최한 해커톤에서 최우수상을 수상하였습니다.
